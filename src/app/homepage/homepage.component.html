What is Medicinish?

Medicinish is an online shop web application that lets users to find their medicine based on the symptoms they have. Medicinish has its own algorithms in the system to find the most suitable medicine for the users by learning on the user's symptoms and contradictions. Medicinish implements "Symptoms Filtering" feature which makes it easier for user to search for their medicine.


What is Symptoms Filtering?

When users are confused on buying which kind of medicine they really need, symptoms filtering feature is the solution to their problem. This feature helps user to find the medicine they need by learning on the user's symptoms. The first step that user needs to do is just listing the symptoms that they had. Medicinish have provided user with a list of common symptoms that most people feel when they are sick to make a better user experience of using the symptoms filtering feature. Users are able to list down three different symptoms that they had and also they need to state if they have any contradictions with medicine. Medicinish will provide the list of medicine that are most suitable with user's symptoms in the product list.
